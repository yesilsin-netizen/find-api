## 🚀 快速开始

### 安装

```bash
# 推荐：通过 Skills CLI 全局安装
npx skills add yesilsin-netizen/find-api -g

# 或：通过 Git 克隆
git clone https://github.com/yesilsin-netizen/find-api.git

# 或：通过 ClawHub CLI
clawhub install find-api
```

### 验证安装

```bash
# 检查是否安装成功
npx skills list -g | grep find-api
```

### 快速使用

本技能会在以下情况自动触发：
- Agent 需要获取外部数据
- 用户提到 "API"、"数据源"、"可靠"
- 网页爬取效率低或失败

## 📋 覆盖领域

| 领域 | 推荐数据源 | 成本 | 认证难度 |
|------|-----------|------|---------|
| **股票/金融** | AKShare, Tushare, yfinance | 免费/付费 | 低 |
| **天气** | OpenWeatherMap, 和风天气 | 免费/付费 | 低 |
| **新闻** | NewsAPI, GNews | 免费/付费 | 低 |
| **地图** | Nominatim, 高德地图 | 免费/付费 | 低 |
| **电商** | 淘宝/京东开放平台 | 免费 | 中 |

## 💡 为什么使用 API？

| 指标 | 网页爬取 | 使用 API | 提升 |
|------|---------|---------|------|
| **速度** | 30-60 秒 | 1-3 秒 | ⚡ 20倍+ |
| **Token 成本** | 高 | 低 | 💰 80%+ |
| **准确性** | 低 | 高 | ✅ 99%+ |
| **稳定性** | 低 | 高 | 🔒 稳定 |
| **维护成本** | 高 | 低 | 🔧 省心 |
